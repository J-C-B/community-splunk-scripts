Quick Start

  • Launch a fresh Ubuntu host and run the script as root or with sudo.
  • Set the seed password variable in the script before running in production.
  • After install, open the printed Splunk URL and verify host banner/hostname.
These scripts are community focused and not supported or endorsed by Splunk. Test in lower environments first.
